ALGORITHM OVERVIEW
══════════════════

For each street:
  1. Find all EXACT matches (house numbers with precise coords)
  2. Sort by house number
  3. For each STREET-LEVEL match:
     - Find nearest EXACT matches (one below, one above)
     - Calculate position: (target - lower) / (upper - lower)
     - Interpolate: coord = lower + position × (upper - lower)
     - Upgrade quality: STREET → EXACT

Requirements:
  ✓ Minimum 2 exact matches per street
  ✓ Numeric house numbers
  ✓ Same settlement and street name
